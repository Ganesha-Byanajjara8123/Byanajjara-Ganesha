## 👨‍💻 About Me

I'm an Electronics and Communication Engineering graduate focused on
**RTL Design** and **Digital Hardware Development**.

I enjoy designing synthesizable RTL in **Verilog**, understanding
processor microarchitecture, and building complete digital systems from
individual hardware blocks.

I'm currently documenting my journey of building a **5-Stage Pipelined
RISC-V Processor**, starting from architecture, designing every RTL
module, integrating a Single-Cycle Processor, and finally implementing
Hazard Detection, Data Forwarding, and Pipeline stages.
